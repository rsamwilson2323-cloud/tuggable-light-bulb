# 💡 Tuggable Light Bulb

An interactive web application that simulates a realistic pull-cord light bulb. Users can pull the hanging cord to switch the bulb ON and OFF, creating an engaging and visually appealing experience. Built with HTML, CSS, and JavaScript, the project demonstrates interactive animations, DOM manipulation, and responsive design principles.

## ✨ Features

* 💡 Realistic light bulb simulation
* 🎯 Interactive pull-cord mechanism
* 🌙 Light ON/OFF functionality
* ✨ Smooth visual transitions
* 🎨 Modern and clean UI design
* 📱 Mobile-friendly experience
* 🖱️ Mouse and touch interaction support
* ⚡ Fast and lightweight performance
* 🚀 No external dependencies

## 🎮 How to Use

1. Open the application in your browser.
2. Locate the hanging pull cord.
3. Click, drag, or pull the cord.
4. Watch the light bulb switch ON or OFF.
5. Enjoy the interactive lighting effects.
